## 🚀 Thông tin project

### 1. Game 2D (Python + Pygame)

Đây là game 2D được viết hoàn toàn bằng **Python** sử dụng thư viện **Pygame**.

- Code tay 100% (không dùng engine như Unity, Godot)

- Có xử lý:
  - Game loop
  - Input bàn phím
  - Collision (va chạm)
  - Rendering (hiển thị)

---

### 2. Cách chạy game

Cài thư viện:

```bash
pip install pygame
```

Chạy game:

python src/main.py
3. Điều khiển
SPACE: Nhảy
WADS: Di chuyển
ESC: Thoát game
